# PDA Παραγγελιοληψία

Δωρεάν εφαρμογή παραγγελιοληψίας (PDA) για καφετέρια/εστιατόριο. PWA — τρέχει σε Android και iOS χωρίς app store, χωρίς συνδρομές.

## Τεχνολογίες

- **Frontend**: React + TypeScript + Vite, PWA (εγκατάσταση στην αρχική οθόνη)
- **Backend**: Supabase free tier (Postgres + Realtime + Auth)
- **Hosting**: GitHub Pages (δωρεάν)

## Λειτουργίες

- Τραπέζια με ζωντανή κατάσταση (ελεύθερο/ανοιχτό + σύνολο) — συγχρονισμός σε όλες τις συσκευές μέσω Realtime
- Παραγγελία ανά τραπέζι: κατηγορίες, προϊόντα, ποσότητες, σημειώσεις, προσθήκες σε ανοιχτό τραπέζι
- Εκτύπωση δελτίου παραγγελίας και λογαριασμού σε χαρτί 80mm μέσω `window.print()`
- Διαχείριση μενού/τραπεζιών από την εφαρμογή
- Ημερήσιες αναφορές (τζίρος, παραγγελίες, πωλήσεις ανά προϊόν)

> Τα δελτία **δεν αποτελούν φορολογικές αποδείξεις** (δεν αντικαθιστούν ΦΗΜ/myDATA).

## Εγκατάσταση στα κινητά

1. Άνοιξε το URL της εφαρμογής στον browser του κινητού.
2. **Android (Chrome)**: μενού ⋮ → «Προσθήκη στην αρχική οθόνη».
3. **iOS (Safari)**: κουμπί κοινοποίησης → «Προσθήκη στην αρχική οθόνη».
4. Σύνδεση με τον κοινό λογαριασμό του μαγαζιού — μένει συνδεδεμένο.

## Ανάπτυξη

```bash
npm install
npm run dev
```

Σύνδεση Supabase: ορίζεται στο `src/lib/supabase.ts` (ή μέσω `VITE_SUPABASE_URL` / `VITE_SUPABASE_KEY`).

## Deploy

Κάθε push στο `main` κάνει αυτόματο deploy σε GitHub Pages μέσω GitHub Actions.
